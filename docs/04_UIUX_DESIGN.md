# Remind Link - UI/UX 디자인 가이드

## 1. 디자인 원칙

### 1.1 핵심 가치
- **발견의 즐거움**: "아, 내가 이런 글도 저장했었구나"를 느낄 수 있게
- **시각적 매력**: 보기 좋은 카드 디자인으로 콘텐츠를 매력적으로 표시
- **직관적 사용**: 복잡하지 않고 직관적인 인터페이스
- **빠른 탐색**: 원하는 콘텐츠를 빠르게 찾을 수 있게

### 1.2 디자인 철학
- **미니멀리즘**: 불필요한 요소 제거, 콘텐츠에 집중
- **카드 기반**: 각 콘텐츠를 독립적인 카드로 표현
- **색상 활용**: 카테고리별 색상으로 시각적 구분
- **애니메이션**: 부드러운 전환 효과로 사용자 경험 향상

## 5. 주요 컴포넌트

#### 카드 뷰
```
┌─────────────────────────────────────┐
│  [이미지]                            │
│  ┌───────────────────────────────┐  │
│  │ [카테고리 배지]                │  │
│  └───────────────────────────────┘  │
│                                      │
│  제목 (Bold, 18px)                   │
│  설명 (Regular, 14px, 2줄 말줄임)    │
│                                      │
│  [태그1] [태그2] [태그3]             │
│                                      │
│  [플랫폼 아이콘] • 3일 전 • 5분 읽기 │

```

**상태**:
- **기본**: 그림자 없음, 호버 시 그림자 추가
- **호버**: 카드가 살짝 올라오는 효과 (transform: translateY(-4px))
- **클릭**: 링크로 이동 또는 모달 열기

#### 리스트 뷰
```
┌─────────────────────────────────────────────┐
│ [이미지]  제목                    [플랫폼]   │

│           [태그1] [태그2]                    │
```

## 6. 인터랙션 디자인

### 6.1 애니메이션

#### 페이드 인
```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
```

#### 슬라이드 업
```css
@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

#### 스태거 애니메이션 (카드 목록)
- 각 카드가 순차적으로 나타남
- 0.1초 간격으로 딜레이

### 6.2 호버 효과

#### 카드 호버
- `transform: translateY(-4px)`
- `box-shadow` 증가
- `transition: all 0.2s ease`

#### 버튼 호버
- 배경색 변경
- 스케일 약간 증가 (1.02)

### 6.3 로딩 상태

#### 스켈레톤 UI

## 8. 접근성 (A11y)

### 8.1 키보드 네비게이션
- Tab으로 모든 인터랙티브 요소 접근 가능

### 8.2 스크린 리더
