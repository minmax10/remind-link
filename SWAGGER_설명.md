# Swagger가 뭔가요?

## Swagger란?

**Swagger**는 API를 테스트하고 문서를 확인할 수 있는 웹 인터페이스입니다.

### 용도
- API 문서 확인
- API 직접 테스트
- 요청/응답 형식 확인

### 하지만...

**프론트엔드가 있으면 Swagger는 필요 없습니다!**

## Remind Link 사용 방법

### ✅ 권장 방법: 프론트엔드 사용

1. **로그인 페이지 사용**
   - http://localhost:3000/login 접속
   - 회원가입 또는 로그인
   - 자동으로 토큰이 저장됩니다

2. **메인 페이지 사용**
   - http://localhost:3000 접속
   - 인스타그램 연동
   - 동기화 버튼으로 저장된 게시물 가져오기

### ❌ Swagger 사용 (선택사항)

Swagger는 개발자가 API를 테스트할 때만 사용합니다.
- 일반 사용자는 사용할 필요 없습니다
- 오류가 나도 무시해도 됩니다
- 프론트엔드에서 모든 기능을 사용할 수 있습니다

## Swagger 오류가 나는 이유

Swagger는 API 테스트 도구이므로:
- 인증 설정이 복잡할 수 있습니다
- 브라우저 호환성 문제가 있을 수 있습니다
- 프론트엔드가 있으면 굳이 사용할 필요가 없습니다

## 결론

**Swagger는 무시하세요!**

프론트엔드(http://localhost:3000)에서 모든 기능을 사용할 수 있습니다:
- ✅ 회원가입/로그인
- ✅ 인스타그램 연동
- ✅ 저장된 게시물 보기

Swagger는 개발자용 도구일 뿐입니다.
